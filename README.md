VitalSync is a full-stack web application built to manage healthcare-related services such as patient records, appointment scheduling, and health tracking in a centralized system. The project includes secure authentication, REST API integration, responsive frontend development, and database operations.

Technologies used include React.js, Node.js, Express.js, MongoDB, and JWT authentication. I contributed to frontend development, backend API creation, state management, and deployment optimization while ensuring scalability and performance.
